tooltap - a library / framework / server for web browser tools

tooltap is a web server that provides a platform for web browser tooling

    https://github.com/pmuellr/tooltap

usage:

    tooltap <options>

port is a tcp/ip port to run the web server on.

options are one of the following:

    -p --port      http port to use (default: 3000)
    -h --help      print help
    -v --verbose   run verbosely
    --version      print the version

tooltat data is stored in ~/.tooltap
